# java-book-market
Projeto Java WEB para lojas de livros.

```
  Para usar o site é preciso realizar o cadastro do usuário
  posteriormente realizar o login
  
  o site contém as seguintes rotas:
  
  /Index
  /Login
  /Users
  /Produtos
  
```
